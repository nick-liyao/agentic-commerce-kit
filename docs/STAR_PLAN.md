# 501-Star Plan

Goal: make Agentic Commerce Kit a credible founder-led open-source project with 501+ real GitHub stars.

## Positioning

Short name:

Agentic Commerce Kit

Repository:

`nick-liyao/agentic-commerce-kit`

Tagline:

AI-agent readiness audits and MCP-ready commerce utilities for Shopify and ecommerce stores.

Core story:

Ecommerce stores are optimized for human browsing and legacy feeds. AI agents need reliable, machine-readable product context. Agentic Commerce Kit helps developers audit and expose that context safely.

## Why This Name

- Uses the high-growth term `agentic commerce`.
- Clear enough for GitHub search.
- Not tied to PixVoices or any personal brand.
- Broad enough to support Shopify, headless commerce, MCP, feeds, and audits.
- Compatible with future npm publishing.

## What Not To Do

- Do not buy stars.
- Do not join star-exchange groups.
- Do not rename a copied project and present it as original.
- Do not hide licenses or attribution.
- Do not commit private `.env` files or credentials.

## Evidence To Preserve For OPC

- Public repository URL.
- Commit history showing founder-led development.
- Release history.
- Issues and roadmap.
- Public launch posts.
- Star history screenshot after milestones.
- Evidence that the project is AI/agent/ecommerce infrastructure, not a simple store utility.
