# Publishing Guide

This project is designed to be published as a public GitHub repository.

## Recommended Repository

```text
nick-liyao/agentic-commerce-kit
```

Recommended project name:

```text
Agentic Commerce Kit
```

## Repository Settings

- Visibility: Public
- License: MIT
- Topics:
  - `mcp`
  - `shopify`
  - `ecommerce`
  - `ai-agent`
  - `vibe-coding`
  - `commerce`
  - `seo`
  - `llms`

## First Release Checklist

- README explains the problem in the first screen.
- CLI command works locally.
- No `.env` files are committed.
- No PixVoices private credentials are committed.
- License is included.
- Example output is included.
- GitHub description is short and search-friendly.

Suggested GitHub description:

```text
AI-agent readiness audits and MCP-ready commerce utilities for Shopify and ecommerce stores.
```

Suggested website field:

```text
https://github.com/nick-liyao/agentic-commerce-kit
```
